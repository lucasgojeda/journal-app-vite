Module: Thunk-Auth

En este thunk encontramos las acciones relacionadas a la autenticación de la aplicación.

Members

static constant startGoogleLogin

Esta función abre una ventana emergente para que el usuario pueda iniciar sesión con su cuenta de google.

static constant startLogoutAction

La siguiente función realiza el cierre de sesión limpiando los datos de usuario y de las notas de el estado en redux.
