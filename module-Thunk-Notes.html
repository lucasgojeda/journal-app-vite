Module: Thunk-Notes

En este thunk encontramos las acciones relacionadas a las notas de la aplicación.

Methods

async static startDeleting(id)

Esta función se encarga de eliminar una nota especifica en base al id enviado.

Parameters:
Name Type Description
id String id de la nota.

async static startLoadingNotes(uid)

Esta función se encarga de cargar las notas al momento de que el usuario inicia sesión.

Parameters:
Name Type Description
uid String uid proporcionado por firebase personal de cada usuario.

async static startNewNote()

Esta función se encarga de crear una nueva nota.

async static startSaveNote(note)

Esta función se encarga de guardar la nota anteriormente creada pero ahora con información en la base de datos.

Parameters:
Name Type Description
note Object Nota a ser guardada.
Example
{
            title: String,
            body: String,
            date: Date,
            url: String,
            id: String
        }

async static startUploading(file)

Esta función se encarga de iniciar el procedimiento de subida de la imagen a cloudinary.

Parameters:
Name Type Description
file file Imagen a ser subida.
